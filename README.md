This project demonstrates collaborative development and machine learning workflow using GitHub and Python. Team members work together through branches, pull requests, and continuous code reviews to improve the project collaboratively.

Key Highlights:

* Collaborate using GitHub branches and pull requests
* Review and improve each other’s code through “ping-pong” updates
* Train a simple machine learning model
* Evaluate model performance on train and test datasets
* Generate and save Actual vs Predicted plots for both training and testing data
* Practice teamwork, version control, and ML model evaluation workflows
